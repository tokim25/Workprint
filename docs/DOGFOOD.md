# Workprint Dogfood Log

This log captures improvements discovered by using Workprint on real projects.

## Investigation INV-0001

**Project:** Workprint

### Goal
Determine whether Workprint can accurately reconstruct its own creation using only available evidence.

### What worked

- Evidence-first workflow prevented unsupported claims.
- Timeline reconstruction was straightforward from conversation history.
- Activity attribution was more useful than percentage attribution.
- Unknowns were surfaced naturally.

### What felt awkward

- Suggestions and final decisions were mixed together.
- Product decisions deserved first-class treatment.
- Evidence coverage was discussed but not summarized consistently.

### New backlog items

1. Add a decision model.
2. Expand the activity taxonomy.
3. Add an evidence coverage summary.
4. Create investigation IDs.
