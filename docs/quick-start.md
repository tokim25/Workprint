# Quick Start

## Install

For Claude Skills, upload the standalone `workprint-claude-skill-v0.1.zip` file.

For Claude Code, preserve the repository's `.claude` directory in the project where you want to use Workprint.

## Gather evidence

Useful sources include AI conversation exports, Git history, document revisions, Figma history, tickets, calendars, time trackers, build logs, meeting notes, and final artifacts.

## Start with validation

```text
/workprint validate the evidence in this folder and tell me what else would materially improve the analysis.
```

Then run `timeline`, `attribution`, `portfolio`, `executive`, or `detailed` as needed.
