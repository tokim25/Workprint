# Attribution Model

Workprint attributes discrete activities and artifacts, not total project ownership.

Labels: Human, named human, named AI system, multiple AI systems, mixed human and AI, automated system, unknown.

Activities: problem definition, research, requirements, planning, architecture, drafting, implementation, editing, testing, debugging, review, decision-making, project management, documentation, deployment, final approval.

Direction, generation, selection, editing, validation, and approval are different contributions and should be recorded separately when evidence permits.
