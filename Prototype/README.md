# Workprint Working Model

A lightweight, dependency-free prototype of the approved four-screen Workprint experience.

## What it includes

1. Start
2. Add where the work happened
3. Investigating
4. Discoveries
5. Supporting evidence dialog
6. Explicit confidence and unknowns

## Run locally

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Important limitations

This is an interaction prototype. It:

- uses sample evidence
- does not upload or process files
- does not run repository analysis
- does not determine authorship, ownership, effort, or contribution percentages
- does not represent a production-ready analysis engine

## Product promise

> Workprint shows what you contributed, what AI contributed, how you worked together, and what the evidence cannot determine.

## Design direction

Warm Investigator: calm, spacious, personal, evidence-driven, and honest about uncertainty.
